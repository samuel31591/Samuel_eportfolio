Network and Information Security Management

  • Samuel Tolofari
  • date:March 2022

This module provides the background knowledge in security concepts and terminology, illustrated by hands on network technology., providing knowledge in the basics of both information security and network technology. Using network technology to illustrate some very important security concepts such as vulnerability assessments, penetration testing, forensic analysis and breach management.

The module will be concluded with a look at some of the challenges that face the Internet, and a consideration of how recent and emerging network technology can be used to mitigate some of the issues presented.

The module focuses on

  • understanding of the fundamentals of computer networks and security.

  • understanding of the principles of information security management.

  • understanding of the use and benefits of various monitoring and logging tools.

  • understanding of how to create and use security vulnerability and assessment tools.

  • understanding of future network architectures and information security management.

  • The opportunity to reflect on and evaluate on personal development.

  • The ability to present critical arguments for specific actions or outcomes to a diverse audience.
